# Guion de Exposición Breve (2-3 minutos)

## 1. Apertura (20-30 segundos)
Buenos días. En este trabajo presento el análisis del ciclo de vida de la aplicación móvil **HotelFlow**, enfocada en reservas de hoteles.  
La app incluye cuatro pantallas principales: Home, Detail, Favorites y Profile.

## 2. Pantallas y objetivo visual (40-50 segundos)
La propuesta de diseño usa una estética premium para transmitir confianza y calidad.  
En Home se prioriza exploración rápida; en Detail se concentra la información clave para decidir; en Favorites se facilita recuperar hoteles guardados; y en Profile se centralizan estadísticas y preferencias.

## 3. Ciclo de vida y relación con UI (60-70 segundos)
- **Planificación y requerimientos:** definimos funcionalidades esenciales y experiencia esperada.
- **Diseño de interfaz y arquitectura:** construimos una estructura modular y una identidad visual consistente.
- **Desarrollo:** implementamos pantallas y navegación funcional.
- **Pruebas:** validamos flujo, estados y consistencia visual.
- **Despliegue/mantenimiento:** se prepara la evolución del sistema con mejoras continuas.

La UI se relaciona con todas las fases porque cada decisión técnica impacta en usabilidad y coherencia visual.

## 4. Cierre (20-30 segundos)
Como conclusión, el proyecto demuestra que un buen diseño de interfaz depende de un ciclo de desarrollo completo, no solo de la parte visual.  
Gracias.
